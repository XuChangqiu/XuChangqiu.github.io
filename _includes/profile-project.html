AlgoVerse: A Personalized Programming Learning Platform Powered by Large Language Models
Project Leader   |   Nov. 2024 - Jul. 2025 项目负责人   |   2024.11 - 2025.07
AlgoVerse is a personalized programming learning platform built with Vue3, Spring Boot, Docker, and PostgreSQL. The system integrates problem recommendation, process mining, learning risk prediction, AI-assisted code evaluation, anti-cheating workflows, knowledge graphs, and ECharts-based learning analytics. AlgoVerse 是一个个性化编程学习平台，基于 Vue3、Spring Boot、Docker 和 PostgreSQL 构建。系统整合题目推荐、过程挖掘、学习风险预测、AI 辅助代码评测、反作弊流程、知识图谱与 ECharts 学情分析等功能。

TCM Diagnostic System Integrating Deep Learning and IoT Simulation 融合深度学习与物联网仿真的中医诊断系统
Team Member   |   IoT Design Competition, Virtual Simulation Application Track 团队成员   |   物联网设计大赛虚拟仿真应用赛项
This project explores a smart traditional Chinese medicine diagnostic system that integrates IoT sensing, virtual simulation, and deep learning. It collects multi-dimensional health indicators such as blood oxygen, blood pressure, heart rate, body temperature, respiration, and tongue-coating images, then combines sensor fusion with AI-assisted analysis to support preliminary health assessment and personalized wellness suggestions. 该项目探索融合物联网感知、虚拟仿真与深度学习的智慧中医诊断系统，采集血氧、血压、心率、体温、呼吸、舌苔图像等多维健康指标，并结合传感器融合与 AI 辅助分析，支持初步健康评估与个性化康养建议。

The system design includes wearable sensing modules, abnormal-condition alerts, safety-threshold configuration, real-time health parameter panels, tongue-coating image analysis, and comprehensive diagnostic suggestions. The tongue-coating module uses transfer learning with a ResNet18-based model to support image recognition, while the overall workflow aims to connect TCM outpatient service, ward care, teaching, and daily health management scenarios. 系统设计包含可穿戴传感模块、异常状态报警、安全阈值配置、实时健康参数面板、舌苔图像分析与综合诊疗建议等功能。舌苔模块基于 ResNet18 迁移学习模型进行图像识别，整体流程面向中医门诊、病房护理、中医教学与日常健康管理等应用场景。
