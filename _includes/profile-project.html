AlgoVerse: A Personalized Programming Learning Platform Powered by Large Language Models
Project Leader   |   Nov. 2024 - Jul. 2025
AlgoVerse is a personalized programming learning platform built with Vue3, Spring Boot, Docker, and PostgreSQL. The system integrates problem recommendation, process mining, learning risk prediction, AI-assisted code evaluation, anti-cheating workflows, knowledge graphs, and ECharts-based learning analytics.

TCM Diagnostic System Integrating Deep Learning and IoT Simulation
Team Member   |   IoT Design Competition, Virtual Simulation Application Track
This project explores a smart traditional Chinese medicine diagnostic system that integrates IoT sensing, virtual simulation, and deep learning. It collects multi-dimensional health indicators such as blood oxygen, blood pressure, heart rate, body temperature, respiration, and tongue-coating images, then combines sensor fusion with AI-assisted analysis to support preliminary health assessment and personalized wellness suggestions.
Show details
The system design includes wearable sensing modules, abnormal-condition alerts, safety-threshold configuration, real-time health parameter panels, tongue-coating image analysis, and comprehensive diagnostic suggestions. The tongue-coating module uses transfer learning with a ResNet18-based model to support image recognition, while the overall workflow aims to connect TCM outpatient service, ward care, teaching, and daily health management scenarios.
