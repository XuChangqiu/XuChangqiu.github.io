AlgoVerse: A Personalized Programming Learning Platform Powered by Large Language Models
Project Leader   |   Nov. 2024 - Jul. 2025
AlgoVerse is a personalized programming learning platform built with Vue3, Spring Boot, Docker, and PostgreSQL. The system integrates problem recommendation, process mining, learning risk prediction, AI-assisted code evaluation, anti-cheating workflows, knowledge graphs, and ECharts-based learning analytics.
